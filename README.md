# 🚲 Bike Sharing Data Analysis

## Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis dataset Bike Sharing untuk memahami pola penyewaan sepeda berdasarkan waktu, cuaca, dan faktor lainnya.

---

## Insight Utama

### 1. Pola Berdasarkan Jam
- Penyewaan tertinggi terjadi pada jam 08.00 dan 17.00
- Hal ini menunjukkan pola commuting (berangkat & pulang kerja)

### 2. Pengaruh Cuaca
- Cuaca cerah (Clear) memiliki jumlah penyewaan tertinggi
- Cuaca buruk seperti hujan menurunkan jumlah penyewaan

### 3. Weekday vs Weekend
- Weekday: tinggi saat jam kerja
- Weekend: lebih stabil sepanjang hari
